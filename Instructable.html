Instructable

The Vanessa is a product for your phone that is used to prevent morning phone usage through creative integrated solutions. This instructable will go through the process of making your own!

Materials Needed

Physical Materials
  1. Alarm System
    • Digital interface microcontroller (e.g., Arduino, Raspberry Pi)
    • Wi-Fi or Bluetooth module for communication
  2. Locking Box
    • A box printed by a 3d printer
    • Electronic lock (e.g., solenoid lock or servo motor)
    • Microcontroller to control the lock
  3. Weight Sensor
    • Load cell and amplifier module
  4. Others
    • Toothbrush
    • 3D printer and filament
    • Caliper (for precise measurements)
Digital Tools
  • Hardware Design
    1. Fusion 360: Design the box and fit-in components like the lock and weight sensor
  • Software Development
    1. App Development
      • Use a cross-platform framework (e.g., Flutter or React Native) to build a smartphone app
      • Integrate hardware control using available APIs/SDKs
      • Backend support (e.g., AWS, Firebase) for storing user preferences and logs
    2. Microcontroller Programming
      • Write firmware in Python (for Raspberry Pi) or C++ (for Arduino)
    3. App Prototyping
      • Use Figma to design the user interface and workflow

Step By Step Instructions

Assemble the Hardware
  1. Prepare the Box
    • To download the Fusion files, please click here for the box and the lid
    • Measure the size of the phone that will be put into the box
    • Revise the parameters in the prototypes to your sizes, add lock funtion. Customize or add other features to your liking!
  2. Install the Lock
    • Secure the solenoid lock or servo motor to the box
    • Connect it to the microcontroller
  3. Add the Weight Sensor
    • Mount the load cell at a fixed position inside the toothbrush holder
    • Calibrate the weight sensor to detect the weight of a toothbrush
  4. Integrate Electronics
    • Connect the lock, weight sensor, and other components to the microcontroller
    • Test communication between all hardware components
Develop the Software
The prototype of the App can be found here
  1. Set Alarm and Lock Settings
    • Allow the user to configure the alarm time and the locking mechanism via the app
  2. Emergency Unlock
    • Provide an option to press the phone's home button for 7 seconds to unlock in case of emergencies
  3. Sync with Hardware
    • Use Bluetooth for real-time communication between the phone and the box
  4. Microcontroller-specific
    • Program the alarm to trigger the microcontroller and lock the box until the weight sensor condition is met
    • Ensure the box onlocks only after detecting an increased weight for a specific minimum time
Workflow for Using the Product
  1. Set the Alarm
    • Open the app and set the alarm time + lock duration
  2. Secure the Phone
    • Place the phone unside the box before sleeping
  3. Morning Routine
    • When the alarm goes off, the phone remains locked in the box. Bursh your teeth and place the toothbrush on the weight sensor.
  4. Unlock the Box
    • The box unlocks after the weight sensor detects a toothbrush for the required time
  5. Emergency Protocol
    • Press the phones home button for 7 seconds to unlock the box in emergencies

    • Conduct thorough testing to calibrate the weight sensor and locking mechanism
